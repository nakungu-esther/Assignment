Software Engineering Week One

Posted on June 28, 2024 by Samson

We were taught how the internet works. Devices connect to a local network (like a home Wi-Fi or a company's LAN). The local network connects to a larger network (like an Internet Service Provider, ISP). ISPs connect to regional networks, which connect to national networks. National networks connect to international networks, forming the global internet backbone. Data is transmitted through this network using protocols like TCP/IP, HTTP, and DNS. Data is routed between networks using IP addresses and packet switching. Servers (special computers) store and manage data, like websites, emails, and files. Clients (like web browsers and apps) request data from servers, which respond with the requested information. We were introduced to HTML and how to use different tags such as Headings, Paragraphs, Links, Images, Lists, Tables, and Forms, and we downloaded Live Server.
